Supla version of firware for NAM 0.3.3 from Nettigo:
https://nettigo.pl/products/tagged/NAM-033

With standard Wemos D1 ESP8266 module works all sensors and additional bh1750 light sensor, but there is not enough RAM to send results to https://aqi.eco
![NAM 0.3.3](https://docs.nettigo.pl/i-en/nam/033/nam-033-std-fa/nam-0.3.3-std-29.jpg)

After replacing the standard board with the ESP32 module, everything works. Make sure the module pinouts match the main NAM board. Możesz użyć np. https://nettigo.pl/products/modul-wifi-bluetooth-esp32-d1-mini-usb-c
![Moduł z ESP32, wyprowadzeniami kompatybilnymi z Wemos D1](https://nettigo.pl/system/images/4286/original.jpg?1727209615)
